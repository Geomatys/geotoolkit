RegisteredOperationBrowser demo

This demo illustrates the use of the RegisteredOperationBrowser widget. The tree shown in this applet is the list of registered JAI operations. Consequently this applet provides also a way to explore the JAI operations available on the host machine.

Widget javadoc: RegisteredOperationBrowser
How to use: Browse the tree and look at the description in the bottom of the canvas.
How to make your own: Instantiate RegisteredOperationBrowser.
Prerequites:
  • Java 6
  • Java Advanced Imaging
  • Firefox 3 on Linux or Windows, or Internet Explorer on Windows (there is known issues with Safari, Opera and Firefox on MacOS)
