Maven repository for Geotoolkit binaries and dependencies

This directory contains Geotoolkit binaries and some dependencies not found in the standard public repository. This is a complement (not a replacement) for the above repository. The directory layout is understood by Maven 2, a project management tool used by the Geotoolkit project. Users are free to link their own project to this repository in order to get automatic downloads of Geotoolkit binaries.

This repository contains also GeoTools binaries in which some modules have been deleted and replaced by Maven dependencies to the Geotoolkit equivalent modules. They are provided as a convenience for those who want functionalities not yet ported to Geotoolkit, but without garanteed: some tests are known to fail. Those modified GeoTools modules and JAR file names are prefixed with "gtm-", to distinguish them from the official GeoTools distribution which prefix them with "gt-".
