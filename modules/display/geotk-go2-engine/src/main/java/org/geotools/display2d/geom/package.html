{@link java.awt.Shape} implementations used for the rendering of geographic features. Some classes in this package are just ordinary Java2D shapes (for example {@link org.geotools.renderer.geom.Arrow2D}, which can be used for rendering wind fields). Some other are more elaborated: they extend {@link org.geotools.renderer.geom.Geometry}, which has support for setting the {@linkplain org.geotools.renderer.geom.Geometry#setCoordinateSystem coordinate system}, {@linkplain org.geotools.renderer.geom.Geometry#setRenderingResolution rendering resolution}, etc. Those classes are not designed for spatial analysis or topology: they are not a replacement for JTS, neither an implementation of ISO-19107. They are rather wrapper around arbitrary source of (x,y) coordinates to be rendered.
